# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-16

### Added
- Initial release of FreshRSS Karakeep extension
- "Add to Karakeep" button on all articles
- Configuration UI for Karakeep URL and API token
- Integration with Karakeep API v1 bookmarks endpoint
- Visual feedback for loading, success, and error states
- Dark mode support
- Automatic article marking as read after successful bookmark creation
- Comprehensive documentation (README, INSTALL, CONTRIBUTING)
- Full error handling and validation

### Features
- Creates bookmarks with article URL and title
- Secure API token storage
- Retry functionality on errors
- Loading animations
- Success/error visual feedback
- Browser console error logging for debugging
